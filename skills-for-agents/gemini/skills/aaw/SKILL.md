# AI-Assisted Work (AAW)

This skill provides access to the Work Management system.

## Available Commands

- **/start-work**: Create a new work item with scope, plan, and progress tracking. See [start-work.md](start-work.md).
- **/progress-work**: Continue work on an existing work item. See [progress-work.md](progress-work.md).
- **/work-status**: Check status of active work items. See [work-status.md](work-status.md).
- **/next-task**: Identify the next task to work on. See [next-task.md](next-task.md).
